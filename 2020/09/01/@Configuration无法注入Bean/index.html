@Configuration无法注入Bean

发表于 2020-09-01 更新于 2021-10-22 分类于 SpringBoot Disqus：

报错

代码

排查

  1. 注入Bean的实体类未标记@Component,@Service,@Controller,@Repository。此demo已标记，未解决✖️

  2. 启动类未添加扫描包@ComponentScan，此demo已添加，未解决✖️

  3. 重命名注入的实体类，问题解决✅

    但是这个方案只是治标不治本，无法了解真正的原因。

解决

上述方案3给了我灵感，仔细看应用启动日志发现

要注入的类MyJob已经存在重名的beanName！定睛一看，初始化的JobScheduler的beanName正好是myJob，重命名为public JobScheduler myJobScheduler()后问题解决！坑自己没商量😂
